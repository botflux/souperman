# Webpack template

A simple webpack template

## Features

- Babel,
- SCSS/CSS,
- HTML template,
- HTML interpolation,
- Webp convertion support for JPG & PNG,
- Image support,
- Font support.

## Future features

- PWA

## How to run

```shell
# Install
npm install
```

```shell
# Start the development server
npm run start
```

```shell
# Generate production files
npm run build
```

```shell
# Start tests (tests are not implemented yet)
npm test
```
